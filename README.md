# AreaGeometricFigure
Тестовое задание MindBox

Эта dll рассчитывает площадь треугольника или круга и самостоятельно определяет фигуру. Проверяет треугольник на прямоугольность



Area area = new Area();

Функция Figure вычисляет площадь фигур

Треугольник: area.Figure(значение1, значение2, значение3);

Круг: area.Figure(радиус, 0, 0)

SQL запрос: SELECT `Имя поля продуктов`, `Имя поля категорий` FROM `Имя таблицы`;
